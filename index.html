VScode 延伸模組 Live Server
發表於2020-11-08|網頁前端VScode延伸模組
簡介一個可以讓你在 VScdoe 用 Chrome 開啟本地伺服器並及時預覽網頁的插件。 安裝與使用 選擇 延伸模組 搜尋 “Live Server“，並安裝。 隨意開啟一個 html 並點擊右下角的 GO Live ，就可以看到預覽囉。 之後只要 保存(Ctrl + S) Live Server 就會直接刷新。 自動保存有的人不太喜歡一直按 Ctrl + S 保存。(我自己以經習慣了xD) 開啟 VScode 設定，並將 Files: Auto Save 設定成 “afterDelay“，之後再自行調整 “Files: Auto Save Delay(保存延遲)” 的參數，如圖 500 毫秒(0.5秒)。 正在學習前端的朋友，我們一起加油吧！ 封面來源 ： 夏嶋めも - 「何じっと見てんのよ・・・」
