Hexo 簡介

官網簡介：
Hexo 是一個快速、簡單且強大的網誌框架。Hexo 使用 Markdown（或其他標記語言）解析您的文章，並在幾秒鐘內，透過漂亮的主題產生靜態檔案。

優點：

  • 可以利用 GitHub 搭建免費的網站。
  • 靜態網頁，體積小、性能好。
  • 適合初學者、老鳥。

缺點：

  • 靜態網頁無法及時更新所有內容
  • 需要一直透過 命令提式位元 輸入指令
